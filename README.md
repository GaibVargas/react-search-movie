<h1>Search Movies</h1>
<p>A movies searcher develop with ReactJS.</p>
<p>It was developed to learn ReactJS, and Theme switcher.</p>

<h1>Technologies</h1>
<ul>
  <li>ReactJS</li>
  <li>Styled Components</li>
  <li>Theme switcher</li>
  <li>API TMDB</li>
</ul>

<h1>Run it</h1>
<ol>
  <li>Clone repository</li>
  <li>Get dependencies
    ```cmd
      yarn
    ```
  </li>
  <li>Init project
    ```cmd
      yarn start
    ```
  </li>
</ol>
